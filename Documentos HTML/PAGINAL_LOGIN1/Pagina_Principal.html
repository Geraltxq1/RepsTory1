Informacion Teorica

Que es la Programacion?

Es el proceso de crear un conjunto de instrucciones (código) en un lenguaje de programación para que una computadora realice tareas específicas o resuelva un problema.


Que es HTML?

HTML, o Lenguaje de Marcado de Hipertexto, es el lenguaje estándar para crear páginas web. Define la estructura y el contenido de un sitio web, utilizando etiquetas para organizar y presentar texto, imágenes, videos y otros elementos.


Que es JavaScript?

Es un lenguaje de programación que aporta interactividad y dinamismo a las páginas web, funcionando junto con HTML y CSS para crear experiencias de usuario más ricas. Permite, por ejemplo, hacer que los botones sean clicables, validar formularios, crear animaciones, actualizar contenido en tiempo real y desarrollar aplicaciones completas del lado del cliente (navegador) y del lado del servidor (con Node.js).
