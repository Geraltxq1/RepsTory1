Que es la Programación?

Es el proceso de crear un conjunto de instrucciones en un lenguaje de programación para que una computadora realice tareas específicas o resuelva un problema.

Que es HTML?

HTML, o Lenguaje de Marcado de Hipertexto, es el estándar para crear páginas web. Define la estructura y el contenido de un sitio web.

Que es JavaScript?

Es un lenguaje de programación que aporta interactividad y dinamismo a las páginas web, funcionando junto con HTML y CSS para crear experiencias ricas.
